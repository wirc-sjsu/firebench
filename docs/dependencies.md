# 13. Dependencies

## Required

- geopandas < 2.0
- h5py < 4.0
- hdf5plugin >= 6.0
- matplotlib > 3.8
- numpy < 3.0
- pint < 1.0
- pyproj < 4.0
- rasterio < 2.0
- reportlab < 5.0
- SALib < 2.0
- scipy < 2.0

## Optional
These dependencies are required for advanced use (testing, offline generation of documentation, etc.)

- bandit
- black
- pylint
- pytest
- pytest-cov
- pytest-mock
- shapely
